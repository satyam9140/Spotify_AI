# Spotify Aura - Vercel Prototype

Unofficial Spotify-inspired GenAI product prototype for Open Project 2026.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL shown in the terminal.

## Deploy on Vercel

1. Upload this folder to GitHub.
2. Go to Vercel -> Add New Project.
3. Import the GitHub repo.
4. Framework Preset: Vite.
5. Build Command: `npm run build`.
6. Output Directory: `dist`.
7. Click Deploy and submit the published Vercel URL.

## Prototype scope

- Prompt-to-session GenAI interaction
- Spotify-like mobile UX
- Group Jam Agent
- Trust and privacy guardrails
- Static simulated output; no real Spotify API or user data used
